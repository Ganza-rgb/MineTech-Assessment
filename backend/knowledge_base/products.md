# MineTech Rwanda — Product Suite

## 1. Minetech OS

**Primary operating record layer** optimized for mid and large-scale commercial mines.

### Core Functionality
- **Unified digital operating record:** replaces paper-based shift logs, inspection forms, and incident reports
- **Real-time dashboard:** production metrics, safety status, equipment utilization, compliance scores
- **Department coverage:** 15 operational departments integrated (exploration, grade control, drilling, blasting, loading, hauling, processing, safety, maintenance, geology, survey, environmental, HR, finance, management)
- **Offline-first design:** works in underground environments with no connectivity; syncs when surface link restored

### Key Features
- Digital checklists and inspection workflows
- Real-time production reconciliation
- Safety incident tracking and root-cause analysis
- Maintenance request routing and tracking
- Integration with existing ERP/accounting systems

## 2. Minetech Corp

**Shared digital infrastructure** specifically tailored for artisanal mines and mining cooperatives.

### Core Functionality
- **Lightweight deployment:** optimized for cooperatives with limited IT infrastructure
- **Group management:** multi-site cooperative coordination
- **Simplified compliance:** automated basic regulatory reporting for small-scale operations
- **Market linkage:** connects cooperatives to international buyers via Minetech Trace

### Target Users
- Artisanal mining cooperatives
- Small-scale mining associations
- Community-owned mining operations

## 3. Minetech Trace

**End-to-end chain of custody and mineral due diligence tracking** solution built for international buyers.

### Core Functionality
- **Mineral traceability:** tracks ore from mine site to smelter to refinery
- **Due diligence documentation:** auto-generates OECD-compliant reports
- **Chain of custody:** sealed bag tracking, custody transfer signatures, tamper evidence
- **Buyer portal:** real-time shipment tracking, document verification, compliance status

### Compliance Frameworks
- OECD Due Diligence Guidance
- ITSCI (International Tin Supply Chain Initiative)
- RMI (Responsible Minerals Initiative) templates
- Dodd-Frank Section 1502 reporting

### Workflow
1. Mine site: bag sealed, tagged, logged into system
2. Buyers: scan tag, verify origin, accept custody
3. Transporter: GPS-tracked movement, condition monitoring
4. Smelter: receive confirmation, update chain of custody
5. Refiner: final certification, export documentation

## 4. Minetech Upstream

**Automated pre-deal counterparty and mineral license verification screening tool.**

### Core Functionality
- **Counterparty screening:** checks buyers, transporters, agents against sanctions lists, PEP databases, adverse media
- **License verification:** validates mineral export/import licenses, mining permits, tax compliance
- **Risk scoring:** generates red/yellow/green risk ratings for deals
- **Audit trail:** immutable record of all screening results

### Use Cases
- Pre-deal buyer verification
- Supplier due diligence
- Government compliance checks
- Internal audit and governance

## 5. Minetech Telco

**Specialized network connectivity and edge computing solutions** customized for deep African open pits.

### Core Functionality
- **Underground network:** leaky feeder + private mesh for deep pits
- **Edge computing:** local data processing where connectivity is unreliable
- **Satellite backup:** automatic failover for remote sites
- **IoT device management:** sensor provisioning, OTA updates, health monitoring

### Deployment Scenarios
- Deep open pits (>300m depth)
- Underground tunnels and shafts
- Remote sites with no cellular coverage
- High-interference environments (near heavy machinery)
