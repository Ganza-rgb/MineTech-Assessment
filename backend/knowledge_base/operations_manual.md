# MineTech Rwanda — Operations Manual

## 1. Shift Structure

### Standard Shifts
- **Day shift:** 06:00 – 14:00
- **Swing shift:** 14:00 – 22:00
- **Night shift:** 22:00 – 06:00

### Shift Handover Protocol
1. Outgoing supervisor completes digital handover form in Minetech OS
2. Outstanding issues highlighted: equipment status, safety alerts, production targets
3. Incoming supervisor confirms receipt + acknowledges outstanding items
4. Handover logged with timestamp + both supervisors' RSSB IDs

## 3. Site Zones

### Rutongo Mine Layout
- **Zone A:** Active extraction — high-grade tantalum ore
- **Zone B:** Stripping / waste removal — preparing for next mining block
- **Zone C:** Maintenance park — equipment staging, repair bay, wash bay
- **Shaft 3:** Primary access shaft — 450m depth, main production level
- **Shaft 5:** Secondary access — ventilation raise + emergency egress
- **Stockpile Area:** Ore stockpiles sorted by grade (High/Med/Low)
- **Processing Plant:** Crushing, screening, gravity separation, magnetic separation

### Rwamagana Site
- **Open pit operation:** shallow-depth tin and tungsten
- **Processing:** all-in-one crushing + dense media separation
- **Camp:** 80-worker accommodation, mess hall, medical clinic

## 4. Grade Control Workflow

### Daily Routine
1. **06:30:** Geologist briefs drilling crew on target zones + expected grades
2. **07:00–12:00:** Reverse circulation (RC) drilling + chip sampling
3. **12:00–13:00:** Samples delivered to on-site lab
4. **13:00–15:00:** XRF analysis + fire assay for high-value samples
5. **15:00:** Results uploaded to Minetech OS → block model updated
6. **15:30:** Grade control meeting — excavator routing adjusted based on results
7. **16:00:** Selective loading begins per updated dig instructions

### Grade Control Decisions
- **Grade > 0.8% Ta₂O₅:** High-grade stockpile → direct to processing
- **Grade 0.3–0.8% Ta₂O₅:** Medium-grade stockpile → blended feed
- **Grade < 0.3% Ta₂O₅:** Low-grade stockpile → deferred processing or waste dump

## 5. Emergency Response

### Emergency Roles
- **Mine Manager:** overall command, external coordination (RMB, RSSB, emergency services)
- **Safety Officer:** scene assessment, evacuation coordination, incident documentation
- **Shift Supervisor:** worker accountability, muster point coordination
- **First Aider:** medical response, triage, liaison with clinic

### Communication Tree
1. **Immediate:** Minetech OS SOS alert → all surface supervisors + mine manager
2. **5 minutes:** RMB + RSSB notified if regulatory threshold breached
3. **15 minutes:** Family liaison officer contacts next of kin if injury/fatality
4. **1 hour:** Initial incident report submitted to RMB via Minetech OS

### Evacuation Routes
- **Primary:** Shaft 3 cage → surface → muster point A (north of pit)
- **Secondary:** Shaft 5 ladderway → surface → muster point B (south of pit)
- **Tertiary:** Emergency drift (west wall) → surface road → muster point C
